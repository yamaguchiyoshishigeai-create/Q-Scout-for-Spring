# 履歴資料

この文書は CLI 単体前提の実行フローです。現行実装では Web 経路として `WebPageController` と `SharedAnalysisService` 経由の流れも存在します。

# 最小API設計 v0.1 - 実行フロー

1. Main.main(args)
2. CliApplication.run(args)
3. ArgumentParserがAnalysisRequestを生成
4. ProjectScanner.scan(request)
5. RuleEngine.analyze(projectContext)
6. ScoreCalculator.calculate(analysisResult)
7. ReportGenerator.generate(...)
8. AiMarkdownGenerator.generate(...)
9. ReportArtifactを生成
10. ExecutionSummaryを生成
11. 標準出力へサマリー表示
