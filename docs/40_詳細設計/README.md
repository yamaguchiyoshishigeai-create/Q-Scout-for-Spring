# 40_詳細設計 Guide

`docs/40_詳細設計/` は、`docs/30_基本設計/基本設計.md` で定めた骨格を、個別 UI、個別差分、運用文書、将来の詳細責務文書へ展開する層である。

## 下位フォルダの役割

- `41_詳細設計正本/`
  - 将来の詳細設計正本群の配置先である。現時点では導入準備段階であり、入口文書のみを置く。
- `42_UI差分設計/`
  - 現行 UI 差分設計の本文群を格納する。
- `43_UI運用・Figma/`
  - Figma 運用、静的資産外部化、UI 運用基盤に関する文書を格納する。
- `44_実装指示書/`
  - Codex 向けの差分実装指示書など、作業用文書を格納する。
- `45_整理方針・補助文書/`
  - 40 系整理計画、JavaDoc 導入方針などの補助方針文書を格納する。

## 推奨読書順

1. `45_整理方針・補助文書/40_詳細設計整理フェーズ実行計画.md`
2. `43_UI運用・Figma/Figma反映運用を含む静的資産外部化仕様書_v1.0.md`
3. `42_UI差分設計/デザイン差分修正仕様書_v1.6.md`
4. 必要に応じて `42_UI差分設計/` 内の v1.1〜v1.4 を遡る
5. `43_UI運用・Figma/ユーザー向け_Figma運用作業手順書_v1.2.md`
6. `45_整理方針・補助文書/JavaDoc導入方針書.md`
