# 詳細設計正本

本フォルダは、Q-Scout for Spring における将来の詳細設計正本群を格納するための置き場である。

ここには、`docs/30_基本設計/基本設計.md` を受けて、よりコード近傍の詳細責務、主要クラス責務、主要フロー責務、主要 DTO の意味構造を整理した文書群を今後整備する予定である。JavaDoc 由来、またはそれに準ずる形で責務を正規化した文書は、本フォルダを主筋の配置先とする。

現時点は導入準備段階であり、まだ正式な詳細設計正本群は未配置である。当面の詳細設計差分、UI 運用文書、実装指示書、整理方針文書は、`../42_UI差分設計/`、`../43_UI運用・Figma/`、`../44_実装指示書/`、`../45_整理方針・補助文書/` を参照すること。
