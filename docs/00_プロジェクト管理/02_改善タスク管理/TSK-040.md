# TSK-040 PR本文テンプレート生成支援を整備する

- 区分: 運用 / 自動化 / PR
- 件名: PR本文テンプレート生成支援を整備する
- 状態: 未解決
- 優先度: 中
- 実施主体: ChatGPT(会話内) → ChatGPT(リポジトリ編集)
- 関連箇所: AI_REPO_RESULT証跡チェッカー, PR本文, ChatGPTリポジトリ編集運用ルール, scripts

## 発端
TSK-029の棚卸しで、PR本文作成時に必須項目を漏らさないためのテンプレート生成支援候補が抽出された。

TSK-032の本体実装により証跡チェッカーは追加済みだが、テンプレート生成支援そのものは未実装であるため、TSK-033の残件登録ゲートにより本候補を後続TSKとして登録する。

## 目的
ChatGPT(リポジトリ編集)やCodexがPR本文を作成する際、変更内容、変更対象外、テスト、確認観点、AI_REPO_RESULT雛形を漏れなく作成できるよう支援する。

## 想定支援項目

- TASK_ID指定によるPR本文雛形生成
- 変更内容欄の雛形生成
- 変更対象外欄の雛形生成
- テスト欄の雛形生成
- 確認観点欄の雛形生成
- `AI_REPO_RESULT` 雛形生成
- 解決済み化専用PR用テンプレート生成

## 実装方針

初版は Markdownテンプレートまたは Python CLI を候補とする。

TSK-035のAI_REPO_RESULT証跡チェック導線と重複しないよう、証跡チェックはTSK-035、本文生成支援は本タスクで扱う。

## AI判断を残す範囲

- 実際の変更内容要約
- verification の意味的十分性
- unverified の網羅性
- 確認観点の妥当性

## 完了条件

- PR本文テンプレート生成支援の方式が整理される
- 通常PR用と解決済み化専用PR用のテンプレート方針が整理される
- 必要に応じてテンプレートまたは生成スクリプトが追加される
- AI_REPO_RESULT証跡チェッカーとの役割分担が明確になる
