# TSK-033 スクリプト化候補の残件登録ゲートと定期棚卸しを運用する

- 区分: 運用 / docs / 自動化
- 件名: スクリプト化候補の残件登録ゲートと定期棚卸しを運用する
- 状態: 確認待ち
- 優先度: 高
- 実施主体: ChatGPT(会話内) → ChatGPT(リポジトリ編集)
- 関連箇所: AI運用作業スクリプト化候補棚卸し, 改善タスク管理ルール, 横断運用規程, scripts

## 発端
TSK-029 の棚卸しにより、AI運用作業のプログラム化候補が複数抽出され、初期実装タスクとして TSK-030、TSK-031、TSK-032 が登録された。

しかし、初期3件の実装が完了した時点で、棚卸し文書に残った候補が自動的に次タスクへ進む仕組みがなければ、作業がそこで途切れる可能性がある。

また、今後も運用手順や横断運用規程が追加・修正されるたびに、新しいプログラム化候補が発生する可能性がある。

## 目的
TSK-030〜TSK-032完了後に、棚卸し残件を再評価して後続TSK登録へ接続する。また、運用手順が追加・修正された際に、プログラム化対象の再棚卸しを行う継続ゲートを運用する。

## 実施内容

- TSK-030〜TSK-032が解決済みであることを確認した。
- 棚卸し文書の残件候補を再評価した。
- 残件候補を `今すぐ登録`、`既存TSKへ統合`、`保留`、`不要` の観点で分類した。
- `今すぐ登録` と判断した候補を TSK-037〜TSK-041 として登録した。
- 登録結果を `AI運用作業スクリプト化候補_残件登録結果.md` に記録した。
- 定期棚卸しゲートの運用タイミングを同文書に再記録した。

## 残件登録結果

| 候補 | 判定 | 登録先 |
|---|---|---|
| docsリンク・配置規程チェッカー | 今すぐ登録 | TSK-037 |
| Web/CLI/Maven検証支援スクリプト | 今すぐ登録 | TSK-038 |
| i18nキー整合チェッカー | 今すぐ登録 | TSK-039 |
| PR本文テンプレート生成支援 | 今すぐ登録 | TSK-040 |
| 解決済み化PR生成支援ツール | 今すぐ登録 | TSK-041 |

## 登録しない候補

今回の残件登録ゲートでは、棚卸し文書に明示されていた残件候補をすべて後続TSKとして登録したため、登録しない候補はない。

ただし、各チェッカーの導線・自動化課題は以下へ既に分離済みのため、重複登録しない。

- TSK-030後続: TSK-036
- TSK-031後続: TSK-034
- TSK-032後続: TSK-035

## 定期棚卸しゲート

以下の変更が発生した場合に、プログラム化対象の再棚卸しを行う。

- 運用規程の追加・修正
- Codex連携規程の追加・修正
- 改善タスク管理ルールの追加・修正
- PR証跡ルールの追加・修正
- docs配置・命名規約の追加・修正
- 検証手順やサンプル評価手順の追加・修正
- AI判断ミスまたは手順逸脱の発生

## 完了条件

- TSK-030〜TSK-032完了後に棚卸し残件が再評価される。
- 必要な残件が後続TSKとして登録される。
- 登録しない候補について理由が記録される。
- 運用手順変更時の定期棚卸しゲートが運用される。
- プログラム化計画が初期3件で途切れない状態になる。

## 完了確認

- 登録結果記録: `AI運用作業スクリプト化候補_残件登録結果.md`
