- `mvn test` は `fragments/summary.html:44` の Thymeleaf 式 parse error により失敗している。
- エラー箇所は PR #27 の削除対象ファイルではない。
- 現行のテスト確認、PR 検証、Codex 実行結果確認を阻害するため、優先度は高い。

## 対応方針
- `src/main/resources/templates/fragments/summary.html` の Thymeleaf 式を確認し、parse error の原因を特定する。
- 必要に応じて、Thymeleaf 式をテンプレートとして解釈可能な形へ修正する。
- 日本語 / 英語表示、サンプル表示、通常解析結果表示の既存意図を壊さない。
- 修正後に `mvn test` を実行し、少なくとも当該 parse error が解消していることを確認する。
- `mvn -DskipTests package` も確認する。
