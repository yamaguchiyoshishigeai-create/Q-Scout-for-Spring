- 件名: Public Free版の完成基準を棚卸しする
- 状態: 解決済み
- 優先度: 最優先
- 実施主体: ChatGPT(会話内) → ChatGPT(リポジトリ編集)
- 関連箇所: `README.md`, `docs/00_プロジェクト管理/01_プロジェクト計画/`, Web UI, サンプル評価, CI

## 発端
Public Free版 / Private Pro版の分離運用方針を追加し、Spring Boot 3.5.x 系までを Public Free版の基本対象とした。次に、Public Free版として「無料で試せる範囲はきちんと動く」状態にするための完成基準を明確化する必要があった。

## 現状認識
README、Web UI、サンプル評価、CI、レポート品質、多言語化、ルール詳細ページなど、Public Free版として不足している可能性のある項目が会話中で複数挙がっていたが、体系的な完成基準としてはまだ台帳化・文書化されていなかった。

PR #50 により、Public Free版の完成基準を `docs/00_プロジェクト管理/01_プロジェクト計画/Public-Free版完成基準.md` として追加し、プロジェクト計画索引および docs 索引から辿れるようにした。

## 対応方針
Public Free版の完成基準を棚卸しし、以下を整理する。

- Free版に必須の機能
- Free版で不足している説明・導線
- Pro版に回すべき機能
- READMEに記載すべき範囲
- docsに残すべき範囲
- 完成判定に必要な検証項目

## 実施内容
