# TSK-019 DependabotのSpring Boot major update運用を整理する

- 区分: 運用 / 依存更新
- 件名: DependabotのSpring Boot major update運用を整理する
- 状態: 未解決
- 優先度: 高
- 実施主体: ChatGPT(リポジトリ編集)
- 関連箇所: `.github/dependabot.yml`, Dependabot PR, Public Free版運用方針

## 発端
PR #38 の Spring Boot 4.0.6 直行更新は、Public Free版の対象外として `@dependabot ignore this major version` により close した。今後も同種の major update が発生した場合に、都度判断が散らばらないよう、Dependabot運用を整理する必要がある。

## 現状認識
Public Free版では Spring Boot 3.5.x 系までを対象とし、Spring Boot 4.x は Private Pro版の検証対象とする方針が確定している。一方、Dependabot設定ファイル側で同方針をどこまで明示・制御するかは未整理である。

## 対応方針
以下を確認・整理する。

- `.github/dependabot.yml` の現行設定
- Spring Boot major update を設定で ignore すべきか
- PRコメント運用で十分か
- Public Free版と Private Pro版で依存更新方針をどう分けるか
- 方針を docs へ追記すべきか

必要なら `.github/dependabot.yml` を更新する。

## 完了条件
Spring Boot 4.x など Public Free版対象外の major update に対する Dependabot 運用方針が明確になり、必要に応じて設定または docs に反映される。
