## 発端
Public Free版では Spring Boot 3.5.x 系までを対象とする方針となり、PR #41 で Spring Boot parent を 3.5.13 へ更新した。移行自体は完了したが、Spring Boot 3.5系で確認すべき変更点やQ-Scoutへの影響を簡潔に残す必要がある。

## 現状認識
Spring Boot 3.5系では、3.3でdeprecatedだった一部要素の削除、テスト関連の挙動、設定値の扱いなど、移行時確認観点が存在する。Q-Scoutは比較的依存構成が軽量だが、Web UI、Thymeleaf、i18n、テスト、CLI導線への影響確認は記録価値がある。

## 対応方針
Spring Boot 3.5系移行に関する以下を整理する。

- Q-Scoutに影響し得るSpring Boot 3.5系変更点
- 影響が低いと判断できる領域
- 検証すべき導線
- PR #41後に確認済み / 未確認の範囲
- 将来Spring Boot 4系をPrivate Pro版で検証する際の前提

必要に応じて、docsに移行影響メモを追加する。

## 完了条件
Spring Boot 3.5系移行時の確認観点とQ-Scoutへの影響が文書化され、今後の依存更新やPrivate Pro版検証の前提として参照できる。
