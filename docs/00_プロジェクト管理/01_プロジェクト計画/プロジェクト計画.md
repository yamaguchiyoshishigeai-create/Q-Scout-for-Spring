# プロジェクト計画

この文書は、`docs/00_プロジェクト管理/01_プロジェクト計画/` 配下に置くプロジェクト計画関連文書の入口・索引です。

## このフォルダの位置づけ

このフォルダは、Q-Scout for Spring の公開方針、計画上の前提、将来的な製品展開方針など、プロジェクト全体の進め方に関わる文書を配置する領域です。

詳細な運用規程や作業証跡ルールは、原則として `docs/00_プロジェクト管理/05_横断運用規程/` を参照します。

## 現行文書

- `Public-Free版・Private-Pro版分離運用方針.md`
  - Public リポジトリを Free 試供版兼ポートフォリオとして整備し、商用 Pro 版を別 Private リポジトリで管理するための運用方針です。

## 今後追加する可能性がある文書

今後、以下のような文書が必要になった場合は、このフォルダへの追加を検討します。

- プロジェクト全体のロードマップ
- Free 版完成基準
- Pro 版展開計画
- 公開版と商用版の機能境界整理
- リリース計画
