Q-Scout for Spring

Spring Boot / Maven プロジェクトの zip をアップロードすると、既存 CLI 資産を利用して品質診断を同期実行し、結果と Markdown レポートをその場で確認できます。

現在は Web MVP として、現行対応範囲のプロジェクトを対象に提供しています。

現在の対応範囲

Java17 想定
フレームワークSpring Boot
ビルドツールMaven
リポジトリ構成単一リポジトリ
モジュール構成単一モジュール
入力形式zip
必須ファイルルートに pom.xml
ソース条件Java ソースコードを含むこと
最大サイズ20MB

現時点で非対応

  • Gradle プロジェクト
  • マルチモジュール構成
  • Spring Framework 単体構成
  • Kotlin 主体プロジェクト
  • マルチリポジトリ横断分析

対応範囲外の構成では、正しく判定できない場合があります。
