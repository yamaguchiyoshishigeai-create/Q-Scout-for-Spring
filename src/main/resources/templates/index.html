Q-Scout for Spring

Spring Boot / Maven プロジェクトの zip をアップロードすると、既存 CLI 資産を利用して品質診断を同期実行し、その場で結果と Markdown レポートを返します。

現在の対応範囲

  • Java 17 想定
  • Spring Boot プロジェクト
  • Maven プロジェクト
  • 単一リポジトリ
  • 単一モジュール構成

アップロード条件

  • zip 形式のみ
  • ルートに pom.xml を含むこと
  • Java ソースコードを含むこと
  • 最大 20MB

現時点で非対応

  • Gradle プロジェクト
  • マルチモジュール構成
  • Spring Framework 単体構成
  • Kotlin 主体プロジェクト
  • マルチリポジトリ横断分析

補足

現在は同期実行の Web MVP です。上記対応範囲外の構成では、正しく判定できない場合があります。
